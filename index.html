Intro

Solid Engine is a game and application engine with the ambition of making it as easy as possible to render solid objects, quantized as voxels. The most unique part of the engine is a ray tracing renderer, that completely forgoes the classic rasterization pipeline, in favor of path tracing, which naturally provides many desirable qualities, such as realistic lighting and logarithmic time complexity with respect to scene complexity.

Other interesting aspects of the engine include

  • Ray traced UI
  • 3D pointer input device
  • Scene diffing

Current status and roadmap

The engine consists of many parts, of which several are standalone. The different parts are all in different stages of development, and follow separate release plans. The long term goal is to release all parts as open source software.

Current limitations

  • Works only on Linux
  • Works only on Nvidia graphics cards
  • Scenes limited to 1024 voxels in z-direction
